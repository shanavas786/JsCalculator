JsCalc v1.1

JsCalculator is a small javascript calculator written by shanavas <shanavas[dot]m2[at]gmail.com>, and released under GNU General Public License version 2.3 or later.

Suggestions and feedbacks at shanavas.hostzi.com/feedback.php.

Input
Input can be given using JsCalculator keypad. JsCalculator version 1.1 enables input from standard input also.
Operations
In basic mode JsCalculator provides basic arithmetic (addition, subtraction, multiplication, division) and square root operations of numbers.
